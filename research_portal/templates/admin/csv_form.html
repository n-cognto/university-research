CSV Format Requirements:

Your CSV file should have the following columns:

  1. email - Used as username (required)
  2. first_name - First name (required)
  3. last_name - Last name (required)
  4. middle_name - Middle name (optional)
  5. phone_number - Phone number (required)
  6. id_number - ID number (required)
  7. dob - Date of birth (YYYY-MM-DD)
  8. gender - Gender (male/female/other)
  9. location - Location (optional)
  10. bio - Bio text (optional)

Example:

email,first_name,last_name,middle_name,phone_number,id_number,dob,gender,location,bio
john@example.com,John,Doe,A,+123456789,ID12345,1980-01-15,male,New York,"Staff member bio"
